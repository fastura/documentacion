API para el envío de mensajes individuales (chats 1:1) en un entorno multi-tenant.

Incluye todos los tipos de mensajes soportados por WhatsApp:
- Texto simple
- Imágenes, videos, documentos, audios/voz, stickers
- Ubicaciones, contactos, reacciones, encuestas (polls)
- Mensajes interactivos (listas, botones)

Todos los endpoints usan el número de destino (`number` o `remoteJid`) y requieren autenticación Bearer.
