API para la gestión y consulta de llamadas (voz/video) en WhatsApp.

Incluye:
- Obtener información sobre llamadas activas o historial
- Acciones básicas como rechazar o finalizar llamadas (si la instancia lo soporta)

Nota: La mayoría de eventos de llamada se manejan vía webhooks (ver sección Integrations).
