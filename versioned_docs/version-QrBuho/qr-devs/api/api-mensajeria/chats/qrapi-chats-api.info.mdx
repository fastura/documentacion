API para la gestión de chats individuales (conversaciones 1:1) en un entorno multi-tenant.

Endpoints incluidos en este archivo:
- Listar/obtener chats
- Archivar/desarchivar chats
- Marcar como leído
- Eliminar chat
- Cambiar estado de presencia/typing

Nota: Los endpoints de envío de mensajes (texto, imagen, video, etc.) están en la sección "Send Message" separada.
