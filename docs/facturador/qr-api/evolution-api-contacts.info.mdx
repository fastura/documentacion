API para la extracción y gestión de contactos de WhatsApp.

Incluye:
- Obtener lista completa de contactos de una instancia
- Filtrar entre contactos individuales y grupos
- Acceso a información de perfil (nombre, foto, etc.)

**Tipos de identificadores (remoteJid):**
- Contactos: `{numero}@s.whatsapp.net`
- Grupos: `{id_grupo}@g.us`
- Canales: `{id}@lid` (no son números reales)

⚠️ El endpoint retorna TODOS los contactos sin paginación.
