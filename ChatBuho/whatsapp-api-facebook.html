📄️ Actualizaciones de precios basados ​​en conversaciones: WhatsApp API a partir del 1 de Abril de 2023

A las empresas que utilicen la plataforma empresarial de WhatsApp se les cobrará por conversación de 24 horas, con diferentes tarifas por categoría de conversación. Las conversaciones iniciadas por negocios de hoy se dividirán en tres nuevas categorías: utilidad/servicios públicos, autenticación y marketing. Las conversaciones iniciadas por el usuario de hoy se denominarán conversaciones de servicio.

📄️ ¿Cuáles son las pautas para elegir un nombre para mostrar?

Cuando agrega un nuevo número de teléfono a su cuenta de WhatsApp en Facebook Business Manager, le asigna un nombre para mostrar. El nombre para mostrar es necesario para crear un certificado, sin el cual el cliente de la API de WhatsApp Business no funcionará.

📄️ Diferencias entre WhatsApp Business Y WhatsApp API

Si buscas usar WhatsApp para hacer marketing, tienes que conocer la diferencia entre la aplicación básica de WhatsApp, WhatsApp Business y la API de WhatsApp Business. Son tres herramientas distintas, dirigidas a diferentes usuarios.
