API

Documentación de cómo trabajan las api QR Buho

📄️ Introduction

Endpoint para enviar archivos PDF, multimedia y mensajería masiva a través de WhatsApp.

📄️ Enviar mensaje de texto

Permite enviar un mensaje de texto a un número de WhatsApp.

📄️ Enviar multimedia

Permite enviar imágenes, videos o audios a un número de WhatsApp.

📄️ Enviar PDF

Permite enviar un archivo PDF a un número de WhatsApp.
